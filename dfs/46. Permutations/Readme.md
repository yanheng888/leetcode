**解法二：插板法**

比如求[0,1,2]的所有排列，初始化一个[0],然后逐个插入1，2

- ｜0｜ 有两个插入位置，插入 1 则有 [0,1],[1,0]

- |0|1| 有三个插入位置，插入 2 则有 201，021，012

- |1|0| 有三个插入位置，插入 2 则有 210，120，102

最后可得，[0,1,2]的所有排列组合为201，021，012，210，120，102
