完全背包要素：

1. loop 顺序。如果第一层loop遍历coins,则可以避免重复的组合。如果第一层遍历amount则会包含重复的组合。参考377，322，518

2. 初始化细节。有两种情况，

   1. 要求恰好装满背包。F[1...V] = -$\infty$ ,F[0]=0, 这样就保证最终得到的是一种恰好装满背包的最优解
   2. 不要求恰好装满背包。F[0...V] = 0,
   
3. 状态转移方程。
   $$
   f[i][j]=\begin{cases}
   不拿物品i: f[i-1][j]\\
   拿物品i: f[i][j-v_i]+w_i
   \end{cases}
   $$
